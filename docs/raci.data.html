Returns the claims base layer path for a specified table.

Parameters:

  • table_name – The name of the claims table.

  • raci_environment – Optional. The RACI environment context. If None, the default environment will be used.

  • secured – Optional. Indicates whether to read from a secure container. Defaults to False.

  • sub_category – Optional. The sub-category for the schema name. Defaults to None.

Returns:

The constructed claims base layer path for the specified table.

# Example:
from raci.data.baselayer import get_claims_baselayer_path
